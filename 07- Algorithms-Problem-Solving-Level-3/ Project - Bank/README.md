# Smart Banking Suite
## C++ Console Application

Smart Banking Suite is a **C++ console-based banking client management system**
developed as part of the **AbouHadhood Platform** training track
under **Dr. Mohamed Abou Hadhoud**.

The project evolves through multiple versions, where each version
adds new features, improves usability, and refines the overall
code structure and architecture.

---

## Project Versions Overview

| Version | Description | Key Features |
|--------|------------|--------------|
| **Project 3 [Bank 1]** | Initial implementation of core banking client management. | Modular code using structs and enums, file-based data storage, clean console interface. |
| **Project 3.1 [Bank 1] – 1st Extension** | Extension introducing account balance management. | Deposit and withdrawal operations, client balance display, total bank balance overview. |
| **Project 3.2 [Bank 1] – 2nd Extension** | Advanced version simulating real-world banking scenarios. | Role-based user permissions, enhanced transaction handling, input validation, persistent data storage. |

---

## About the Project

Smart Banking Suite aims to simulate a **real-world banking system**
through a console application.

The project follows **clean architecture principles**, with a strong
focus on:

- Modularity
- Separation of concerns
- Maintainability
- Reusability

The system is built using **small, single-responsibility functions**
and relies on **file-based data persistence** to manage clients,
users, and transactions efficiently.
