# ATM System
## C++ Console Application

This repository contains a **fully functional ATM System** developed in C++
as part of the **AbouHadhood Platform** training track
under **Dr. Mohamed Abou Hadhoud**.

The application simulates real-world ATM operations with a strong focus on
usability, clean code, and reliable functionality.

---

## Project Overview

The ATM System is a console-based application designed to simulate
real-world banking transactions while strengthening core C++ skills such as:

- File handling for persistent data storage
- User interaction through a clean command-line interface
- Writing modular and maintainable C++ code
- Robust input validation for secure operations

---

## Key Features
### Project 4 – ATM System

- **Secure Login**  
  Authenticate users using account number and PIN.

- **Quick Withdraw**  
  Withdraw predefined amounts for faster transactions.

- **Normal Withdraw**  
  Custom withdrawal with validation (multiples of 5).

- **Deposit**  
  Add funds with immediate balance updates.

- **Check Balance**  
  View account details instantly.

- **File Handling**  
  Persistent storage using text files to ensure data integrity.
